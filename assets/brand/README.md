# PMI Brand Assets

## Official wordmark package

- Identity: Philip Morris International horizontal corporate wordmark.
- Official source: `https://www.pmi.com/resources/images/default-source/default-album/pmi-logoa4f115bd6c7468f696e2ff0400458fff.svg?sfvrsn=37857db4_0`
- `pmi-wordmark.svg`: original vector retrieved directly from the official PMI corporate source by the publication workflow.
- `pmi-wordmark.png`: transparent high-resolution browser derivative generated from that SVG.
- `pmi-wordmark-hires.png`: transparent PDF-rendering derivative generated from that SVG.
- Use: nominative identification of the target employer in an independent candidate campaign.
- Treatment: locally served; proportions, colors, and composition preserved; no animation, recoloring, distortion, crop, or combination into a candidate logo.

The Process Genome mark outside this directory is candidate-created and is intentionally not presented as PMI identity.
