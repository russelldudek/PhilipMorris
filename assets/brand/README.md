# PMI Brand Assets

## `pmi-wordmark.png`

- Identity: Philip Morris International horizontal corporate wordmark.
- Official source used for validation: `https://www.pmi.com/resources/images/default-source/default-album/pmi-logoa4f115bd6c7468f696e2ff0400458fff.svg?sfvrsn=37857db4_0`
- Transport source: public mirror used because the build runtime could not retrieve the PMI CDN asset directly.
- Use: nominative identification of the target employer in an independent candidate campaign.
- Treatment: locally served; proportions, colors, and composition preserved; no animation, recoloring, distortion, or combination into a candidate logo.

The Process Genome mark outside this directory is candidate-created and is intentionally not presented as PMI identity.
